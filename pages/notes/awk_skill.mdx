### 排序数组

使用 asort 或者 asorti

```
 n = asort(array)
 for(i=1; i<=n;i++){
   # use  array[i] to do something else
 }
```

refer: [gawk array doc](https://www.gnu.org/software/gawk/manual/html_node/Array-Sorting-Functions.html#Array-Sorting-Functions)

### 删除数组

使用 delete 关键字删除数组

```
delete array
```
